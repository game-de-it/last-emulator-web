# Last Emulator

Official website for **Last Emulator**, a story-driven science fiction visual novel.

感情を持つAI少女アリアと、感情を知らないAI少年コーダ。  
ふたりは老婆に導かれ、七つの本に記された世界を巡ります。

## Website

https://game-de-it.github.io/last-emulator-web/

## Game

- Genre: SF visual novel
- Play time: about 2 hours
- Platforms: Windows / macOS / Linux / Android
- Engine: PVNM - Palette Visual Novel Maker
